tipos de preguntas de un examen
question type 1
Fill the blank space with the right hanzi
for example
question: 你有_____个杯子？
answer: 几


question type 2
Answer the question
for example
question：你是哪国人？
answer：我是中国人

question type 3
Organize the characters to get a coherent sentence (make sure to keep words together, and separate word by word by 3 or 4 spaces)
for example
question: 上课　 在　 那个时候　 他们
answer: 那个时候他们在上课

question type 4
translate the sentence from Chinese to Spanish
for example
question: 她要坐出租车回家
answer: ella quiere tomar taxi para regresar a casa

question type 5
translate the sentence from Spanish to Chinese 
for example
question: Ahora son las tres y media
answer: 现在是三点半

question type 6
generate a text using between {length} characters and produce 5 questions
for example:
text:
今天早上我和朋友在学校。我们上课的时候，老师说汉语，学生听和写。下课以后，我们去饭店吃米饭，喝茶。中午我跟哥哥去商店买东西，他买衣服，我买水果。下午我们回家，看电视，读书。晚上我用手机写汉字，弟弟在电脑前面工作。现在几点？八点半。太晚了，我明天要去大学。

questions:
今天早上他们在哪里？
下课以后他们做什么？
哥哥在商店买什么？
弟弟下午在哪里？
现在几点？

answers:
今天早上他们在哪里？
在学校。

下课以后他们做什么？
去饭店吃米饭，喝茶。

哥哥在商店买什么？
买衣服。

弟弟下午在哪里？
在电脑前面工作。

现在几点？
八点半

question type 7
Generate 6-7 characters, write a text of {length} characters.

for example
write a text of {length} characters.
question: 水， 几， 买， 看， 老师，先生
answer:
今天先生在商店，看水。老师来了，也买水。几个学生看到老师和先生买水，他们也想买。老师看学生，说：“你们要几杯水？”学生说：“我们要三杯水。”先生和老师一起笑，看他们买水。
